# Data Analysis Project (R)

## Impact of Smoking on Health Status – SOEP Data Analysis  

This project analyzes the relationship between smoking behavior and self-reported health status using the German Socio-Economic Panel (SOEP) dataset.  

Academic project based on real-world panel data (SOEP).  

### Analysis Includes:
- Data cleaning and merging multiple datasets  
- Handling missing values  
- Statistical analysis (Chi-square test, T-test, regression)  
- Data visualization in R  

### Key Findings:
There is a statistically significant negative relationship between smoking and perceived health status.  

### Tools Used:
- R  
- Statistical Analysis  
- Data Visualization
